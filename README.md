Person GUI is a program designed to manage objects within the Person class hierarchy. Its implementation includes the user of Person, RegisteredPerson, and OCCCPerson objects that utilize OCCCDate objects as well as OCCCDate Exceptions. Users can manage files holding a list of Person objects to demonstrate the class hierarchy's functionality. 
The demo video can be viewed at: https://www.youtube.com/watch?v=wdS3zLYHwYo

Veronica Edwards wrote the Person hierarchy, including Person.java, RegisteredPerson.java, and OCCCPerson.java.
